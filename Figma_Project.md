## I utilized ChatGPT to generate my About and How-To sections

### Prompts

> Write an about section for a web app called Studium that helps students study and complete their homework using AI.
> 
> Make it very short.

> Can you give a short description of how to use the app? You upload an image or file of your notes/homework and it uses that to help you.
> 
> You can also prompt the AI and ask it questions.
